# Brick Value Radar

Nederlandstalige MVP voor een LEGO-investeringsdashboard. De eerste versie toont een Investment Radar met lokale mockdata, zoekfunctie, filters, statusoverzicht en responsive set-cards.

## Starten

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

De data staat voorlopig in `src/data/sets.json`.
